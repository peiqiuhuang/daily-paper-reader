## 摘要
为医学文本摘要等特定领域任务微调大语言模型需要消耗大量的计算资源。参数高效微调（PEFT）方法通过仅更新极小比例的参数，提供了极具前景的替代方案。本文在 PubMed 医学摘要数据集上，针对 Flan-T5 模型系列比较了三种适配方法：低秩自适应（LoRA）、提示微调（Prompt Tuning）和全量微调（Full Fine-Tuning）。通过多次随机种子的实验，我们证明了 LoRA 的表现始终优于全量微调；在 Flan-T5-Large 模型上，LoRA 仅需 0.6% 的可训练参数即可达到 43.52 +/- 0.18 的 ROUGE-1 分数，而全量微调的分数为 40.67 +/- 0.21。敏感性分析探讨了 LoRA 秩（rank）和提示词标记（prompt token）数量的影响。我们的研究结果表明，低秩约束提供了有益的正则化效果，挑战了关于全参数更新必要性的假设。代码可在 https://github.com/eracoding/llm-medical-summarization 获取。

## Abstract
Fine-tuning large language models for domain-specific tasks such as medical text summarization demands substantial computational resources. Parameter-efficient fine-tuning (PEFT) methods offer promising alternatives by updating only a small fraction of parameters. This paper compares three adaptation approaches-Low-Rank Adaptation (LoRA), Prompt Tuning, and Full Fine-Tuning-across the Flan-T5 model family on the PubMed medical summarization dataset. Through experiments with multiple random seeds, we demonstrate that LoRA consistently outperforms full fine-tuning, achieving 43.52 +/- 0.18 ROUGE-1 on Flan-T5-Large with only 0.6% trainable parameters compared to 40.67 +/- 0.21 for full fine-tuning. Sensitivity analyses examine the impact of LoRA rank and prompt token count. Our findings suggest the low-rank constraint provides beneficial regularization, challenging assumptions about the necessity of full parameter updates. Code is available at https://github.com/eracoding/llm-medical-summarization

随着生物医学文献（如 PubMed）以每年超过 150 万篇的速度增长，临床医生和研究人员面临严重的信息过载。虽然大语言模型（LLM）在文本摘要方面表现出色，但针对医疗等专业领域进行**全量微调（Full Fine-Tuning）**面临巨大的挑战：需要极高的 GPU 显存、漫长的训练时间以及高昂的能耗。

本研究的核心意义在于探索**参数高效微调（PEFT）**方法在医疗摘要任务中的可行性与优越性。研究旨在回答：在资源受限的情况下，PEFT 方法（如 LoRA 和 Prompt Tuning）能否在保持甚至超越全量微调性能的同时，显著降低计算成本。

### 2. 论文提出的方法论
论文对比了三种主要的适配策略，均基于 **Flan-T5** 编码器-解码器架构：
*   **低秩自适应 (LoRA):** 核心思想是假设模型微调时的权重更新存在于一个“低秩空间”。它冻结预训练权重 $W$，并在旁路引入两个低秩矩阵 $A$ 和 $B$。在本项目中，LoRA 被应用于 Transformer 块的查询（Query）和值（Value）投影层。
*   **提示微调 (Prompt Tuning):** 在输入序列前添加 20 个可学习的连续向量（软提示词），仅更新这些嵌入向量，保持模型所有原始参数冻结。
*   **全量微调 (Full Fine-Tuning):** 作为基准，更新模型的所有参数。

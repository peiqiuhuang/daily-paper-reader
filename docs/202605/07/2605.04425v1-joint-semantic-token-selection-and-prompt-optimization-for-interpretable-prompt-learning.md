## 摘要
视觉语言模型（如 CLIP）实现了强大的视觉-文本对齐，但在通过连续提示学习进行适配时，往往面临过拟合和可解释性有限的问题。虽然离散提示优化提高了可解释性，但它通常依赖于大型外部模型，导致计算成本高且可扩展性有限。在本文中，我们提出了可解释提示学习（IPL），这是一个在离散语义 Token 选择和连续提示优化之间交替进行的混合框架。具体而言，IPL 将语义 Token 选择建模为一个近似子模优化问题，鼓励选择既符合人类理解又具有语义多样性的 Token。它进一步采用交替优化策略，将离散 Token 选择与连续提示微调相结合，在提高可解释性的同时保持了对下游任务的适配能力。我们的框架是即插即用的，可以与现有的提示学习方法无缝集成。在多个基准测试上的广泛实验表明，IPL 在五种代表性提示学习方法中一致地提高了可解释性和准确性，为现有框架提供了一种有效且可扩展的扩展。

## Abstract
Vision-language models such as CLIP achieve strong visual-textual alignment, but often suffer from overfitting and limited interpretability when adapted through continuous prompt learning. While discrete prompt optimization improves interpretability, it usually depends on large external models, leading to high computational costs and limited scalability. In this paper, we propose Interpretable Prompt Learning (IPL), a hybrid framework that alternates between discrete semantic token selection and continuous prompt optimization. Specifically, IPL formulates semantic token selection as an approximate submodular optimization problem, encouraging tokens that are both human-understandable and semantically diverse. It further adopts an alternating optimization strategy to integrate discrete token selection with continuous prompt tuning, improving interpretability while preserving adaptability to downstream tasks. Our framework is plug-and-play, allowing seamless integration with existing prompt learning methods. Extensive experiments on multiple benchmarks show that IPL consistently improves both interpretability and accuracy across five representative prompt learning methods, providing an effective and scalable extension to existing frameworks.

### 1. 核心问题与整体含义
*   **研究背景**：视觉语言模型（VLM，如 CLIP）在下游任务适配时，主流方法是**连续提示学习**（如 CoOp），即在嵌入空间优化一组“软提示”向量。
*   **核心痛点**：
    1.  **过拟合与泛化差**：连续向量容易在少量训练样本上过拟合，导致在新类别上表现不佳。
    2.  **缺乏可解释性**：优化的嵌入向量通常无法映射回具体的自然语言单词，人类无法理解模型学到了什么。
    3.  **外部依赖重**：现有的离散提示优化方法往往依赖大型语言模型（LLM）或多模态大模型（LMM）来生成提示，计算成本高且难以扩展。
*   **研究目标**：提出一种无需外部大模型、兼顾可解释性与任务适配能力的混合提示学习框架。

### 2. 方法论
*   **核心思想**：提出 **IPL（Interpretable Prompt Learning）** 框架，通过**交替优化策略**，在离散的“语义 Token 选择”和连续的“提示向量优化”之间切换。
*   **关键技术细节**：
    1.  **候选词过滤**：从 NLTK 语料库中筛选出 8,883 个高质量候选词。筛选标准包括：纯字母、长度≥3、属于 WordNet 词库、Zipf 词频得分≥3.5，且在 CLIP 分词器下仅占 1 个 Token。
    2.  **语义 Token 选择（离散阶段）**：将其建模为**近似子模优化（Submodular Optimization）**问题。目标函数结合了**效用项**（衡量加入 Token 后分类损失的降低程度）和**冗余惩罚项**（衡量新 Token 与已选 Token 间的语义相似度）。采用贪心算法逐一挑选最具代表性且多样化的单词。
    3.  **交替优化策略**：
        *   每选择一个离散语义 Token 插入提示模板，就对剩余的连续软提示向量进行 $t$ 轮梯度下降优化。
        *   这种方式让连续向量能够围绕固定的语义锚点进行适配，缓解了早期选择偏差。
    4.  **提示结构**：最终提示由“软提示向量 + 选定的离散 Token + 类别名称”交错组成。

*   **数据集**：涵盖 11 个主流基准测试，包括 ImageNet（通用）、OxfordPets（宠物）、FGVCAircraft（飞机）、EuroSAT（遥感）、DTD（纹理）等。
*   **Benchmark 场景**：
    1.  **基类到新类泛化**：在基类训练，在新类测试，评估调和平均值（HM）。
    2.  **跨数据集评估**：在 ImageNet 训练，直接在其他 10 个数据集测试。
    3.  **领域泛化**：在 ImageNet 训练，在 ImageNet-V2/S/A/R 等变体上测试。

*   **硬件环境**：实验在单张 GPU 上完成。
*   **算力开销**：

    *   **总体评价**：相比于调用外部大模型，IPL 的计算开销非常小，具有良好的可扩展性。

### 5. 实验数量与充分性
*   **实验规模**：在 11 个数据集上针对 5 种基线方法进行了全面测试，每组实验运行 3 次取平均值。
*   **消融实验**：详细探讨了多样性惩罚系数 $\lambda$、Token 数量 $k$、交替间隔 $t$ 以及提示模板设计对性能的影响。
*   **客观性**：引入了人类评估者对选出 Token 的相关性进行打分（1-5 分），并与 CoOp 检索到的近邻词进行对比，定性与定量结合，实验设计较为严谨、公平。

### 6. 主要结论与发现
*   **性能提升**：IPL 在所有基线方法上均实现了 HM 值的提升。在 CoOp 上平均提升了 **4.07%**，在 EuroSAT 的新类上甚至提升了 **8.41%**。
*   **泛化增强**：通过引入固定的语义 Token 作为先验，模型在未见过的数据集和领域偏移下表现出更强的鲁棒性。
*   **可解释性突破**：IPL 选出的单词（如 OxfordPets 中的 "furry", "veterinary"）具有明确的物理或功能含义，而传统方法检索到的近邻词往往是无意义的字符碎片。

### 7. 优点
*   **即插即用**：作为一个通用模块，可以轻松集成到现有的各种提示学习框架中。
*   **无需外部模型**：完全在内部完成语义发现，避免了对 LLM/LMM 的依赖，降低了部署门槛。
*   **理论支撑**：利用子模优化理论来处理离散选择，比随机搜索或启发式搜索更具科学性。

### 8. 不足与局限
*   **语义覆盖偏差**：对于某些语义极其宽泛（如 ImageNet）或类内差异巨大的数据集，选出的 Token 相关性得分较低，说明该方法在处理高度抽象概念时仍有挑战。
*   **模板依赖性**：性能仍受限于手工设计的插入模板（如 "with emphasis on:"），不同的模板会导致选出不同的 Token。
*   **静态选择**：目前的语义 Token 在训练后是固定的，未来若能实现根据输入图像动态调整语义 Token，效果可能会更好。

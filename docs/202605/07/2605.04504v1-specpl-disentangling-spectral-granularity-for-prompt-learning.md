## 摘要
现有的视觉语言模型（VLM）提示学习表现出模态不对称性，主要优化文本标记，而仍依赖冻结的视觉编码器作为整体提取器，忽略了对细粒度判别至关重要的频谱粒度。为弥补这一缺陷，我们提出了用于提示学习的频谱粒度解耦（SpecPL），通过反事实颗粒监督（Counterfactual Granule Supervision）从全新的频谱视角进行提示学习。具体而言，我们利用冻结的 VAE 将视觉信号分解为语义低频段和粒度高频细节。冻结的视觉语义库（Visual Semantic Bank）将文本表示锚定到通用的低频不变性上，从而缓解过拟合。关键在于，细粒度判别由反事实颗粒训练驱动：通过置换高频信号，我们迫使模型显式地将视觉粒度与语义不变性区分开来。独特的是，SpecPL 作为一个通用的即插即用增强器，通过视觉侧引导使 CoOp 和 MaPLe 等面向文本的基准模型焕发活力。在 11 个基准测试上的实验证明了具有竞争力的先进性能，达到了 81.51% 调和平均准确率的新性能上限。这些结果验证了带有反事实监督的频谱解耦有效地弥合了稳定性与泛化性权衡之间的差距。代码已发布于 https://github.com/Mlrac1e/SpecPL-Prompt-Learning。

## Abstract
Existing prompt learning for VLMs exhibits a modality asymmetry, predominantly optimizing text tokens while still relying on frozen visual encoder as holistic extractor and neglecting the spectral granularity essential for fine-grained discrimination. To bridge this, we introduce Disentangling Spectral Granularity for Prompt Learning (SpecPL), which approaches prompt learning from a novel spectral perspective via Counterfactual Granule Supervision. Specifically, we leverage a frozen VAE to decompose visual signals into semantic low-frequency bands and granular high-frequency details. A frozen Visual Semantic Bank anchors text representations to universal low-frequency invariants, mitigating overfitting. Crucially, fine-grained discrimination is driven by counterfactual granule training: by permuting high-frequency signals, we compel the model to explicitly distinguish visual granularity from semantic invariance. Uniquely, SpecPL serves as a universal plug-and-play booster, revitalizing text-oriented baselines like CoOp and MaPLe via visual-side guidance. Experiments on 11 benchmarks demonstrate competitive state-of-the-art performance, achieving a new performance ceiling of 81.51\% harmonic-mean accuracy. These results validate that spectral disentanglement with counterfactual supervision effectively bridges the gap in the stability-generalization trade-off. Code is released at https://github.com/Mlrac1e/SpecPL-Prompt-Learning.

### 1. 核心问题与整体含义
论文指出了现有视觉语言模型（VLM）提示学习（Prompt Learning）中的**模态不对称性**问题：大多数方法过度集中在优化文本端的 Token，而将视觉编码器视为冻结的整体提取器。这种做法忽略了视觉信号中的**频谱粒度**（Spectral Granularity），导致模型难以捕捉细粒度判别信息，并在“稳定性-泛化性”权衡中表现不佳（即容易过拟合训练集，但在新类别上泛化差）。

### 2. 方法论
SpecPL 提出从频谱视角重新审视提示学习，核心思想是将视觉特征分解为**语义不变性（低频）**和**判别颗粒（高频）**。

*   **空间-频谱代理（Spatial-Spectral Proxy）**：利用一个冻结的预训练 VAE（如 Stable Diffusion 的 VAE）作为教师模型。通过滑动平均池化（Sliding Average Pooling）将 VAE 的潜在空间特征分解为：
    *   **Base（基底）**：低频成分，捕捉稳定的类别结构（如形状、拓扑）。
    *   **Detail（细节）**：高频残差，捕捉实例特有的纹理和细粒度线索。
*   **视觉语义库（Visual Semantic Bank, VSB）**：建立一个存储低频视觉原语的库，通过软检索（Soft Retrieval）将文本提示锚定在这些稳定的视觉特征上，缓解文本端的过拟合。
*   **反事实颗粒监督（Counterfactual Granule Supervision）**：
    *   **事实分支**：利用 FiLM 调制层将高频细节注入视觉特征，进行标准分类训练。
    *   **反事实分支**：在训练期间随机置换（Permute）样本间的高频信号，并强制模型预测颗粒的来源标签。这迫使模型意识到细微的纹理变化足以改变身份，从而增强对细粒度证据的敏感性。
*   **即插即用**：SpecPL 仅在训练时需要 VAE 教师和 FiLM 分支，**推理时完全舍弃 VAE**，不增加任何推理延迟。

### 7. 优点
*   **视角新颖**：首次将 VAE 潜在空间的频谱分解引入提示学习，解决了模态不对称问题。
*   **推理零成本**：巧妙的训练-推理分离设计，使其成为一个纯粹的“性能增压器”。
*   **通用性强**：作为插件可提升多种现有方法（CoOp, MaPLe 等）的性能。
*   **反事实机制**：通过置换高频信号来纠正 VLM 的“形状偏见”（Shape Bias），具有很强的解释性。

### 8. 不足与局限
*   **依赖外部教师**：训练阶段必须依赖预训练的 VAE，增加了训练流程的复杂度和内存占用。
*   **粗粒度任务收益递减**：在语义差异极大的粗粒度任务（如标准 ImageNet）上，相比强基准（如 MMRL）的提升较为微弱。
*   **噪声敏感性**：由于显式依赖高频信号，如果输入图像存在严重的 JPEG 压缩伪影或高频噪声，模型可能会将噪声误认为判别特征。
*   **超参数敏感**：反事实损失的权重需要精细调节，否则可能导致训练不稳定。

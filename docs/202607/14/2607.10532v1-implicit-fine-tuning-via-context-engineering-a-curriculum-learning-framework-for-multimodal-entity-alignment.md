## 摘要
多模态实体对齐（MMEA）旨在识别跨不同模态的等价实体。虽然现有方法通过黑盒上下文工程策略提升了 MMEA 的性能，但它们对大语言模型（LLM）参数容量的依赖以及缺乏理论可解释性的问题仍未得到解决。为此，我们首先在理论上验证了 MMEA 任务中上下文工程与模型微调之间的数学等价性，证明了提示词组件模拟了 MMEA 中基于对比学习的序列微调。在此基础上，我们提出了 PTFEA，这是一个受课程学习启发的框架，将微调策略转化为可解释的上下文工程。具体而言，自适应难度调节利用置信度阈值动态调整信息注入阶段，建立了课程学习权重与上下文样本选择之间的数学等价性；而三阶段渐进式推理则将实体信息从简单到复杂案例逐步纳入，镜像了微调中的梯度下降过程。在五个公开数据集上的实验表明，PTFEA 始终优于强基准模型。特别是在 ICWIKI 数据集上，PTFEA 将 Qwen2.5-72B 与 14B 之间的 H@1 差距缩小到了 0.6%。此外，与代表性的基于上下文工程的 MMEA 方法 MM-ChatAlign 相比，PTFEA 将 Qwen2.5-72B 的运行时间从 21 小时缩短至 1 小时，并将 token 消耗从 2200-3000 降低至 200-400，在 ICWIKI 数据集上实现了超过 80% 的削减。本研究提供了首个统一 MMEA 中上下文工程与微调的理论框架，为未来寻求将更多微调策略转化为上下文工程范式的研究铺平了道路。我们的代码可在 https://github.com/DMiC-Lab-HFUT/PTFEA 获取。

## Abstract
Multimodal Entity Alignment (MMEA) aims to identify equivalent entities across different modalities. While existing methods enhance MMEA performance through black-box context engineering strategies, their reliance on LLM parameter capacity and lack of theoretical interpretability remain unresolved. To this end, we first theoretically validate the mathematical equivalence between context engineering and model fine-tuning in MMEA tasks, demonstrating that prompt components simulate contrastive learning-based sequential fine-tuning in MMEA. Building on this foundation, we then propose PTFEA, a curriculum-learning-inspired framework that translates fine-tuning strategies into interpretable context engineering. Specifically, adaptive difficulty modulation dynamically adjusts information injection stages using confidence thresholds, establishing mathematical equivalence between curriculum learning weights and context sample selection; and three-stage progressive inference incorporates entity information from simple to complex cases, mirroring the gradient descent process in fine-tuning. Experiments on five public datasets demonstrate that PTFEA consistently outperforms strong baselines. In particular, on the ICWIKI dataset, PTFEA narrows the H@1 gap between Qwen2.5-72B and 14B to 0.6%. Moreover, compared with the representative context-engineering-based MMEA method MM-ChatAlign, PTFEA reduces the runtime of Qwen2.5-72B from 21 hours to 1 hour and lowers token consumption from 2200-3000 to 200-400, achieving over 80% reduction on the ICWIKI dataset. This work provides the first theoretical framework unifying context engineering and fine-tuning in MMEA, paving the way for future research that seeks to translate additional fine-tuning strategies into context engineering paradigms. Our code is available at https://github.com/DMiC-Lab-HFUT/PTFEA.

## 1. 核心问题与研究动机
*   **核心问题**：多模态实体对齐（MMEA）旨在识别不同知识图谱中跨模态（文本、图像、结构）的等价实体。现有基于大语言模型（LLM）的重排序方法虽然有效，但存在两个痛点：
    1.  **黑盒性与缺乏理论支撑**：上下文工程（Context Engineering, CE）的设计多基于启发式规则，缺乏数学解释性。
    2.  **高昂的计算成本**：处理多模态信息（尤其是图像）会导致 Token 消耗巨大且推理速度极慢，且小规模 LLM 在复杂提示词下的表现远逊于大规模模型。
*   **研究背景**：作者试图通过理论证明“提示词工程即微调”，并引入课程学习（Curriculum Learning）思想，构建一个高效、可解释且对模型规模鲁棒的对齐框架。

## 2. 方法论：PTFEA 框架
*   **核心思想**：将上下文工程视为一种“隐式微调”。通过数学推导证明，LLM 的 Softmax 注意力机制在处理结构化提示词时，等价于在双模型上进行一步梯度下降。
*   **关键技术细节**：
    *   **理论等价性证明**：扩展了前人研究，证明在 MMEA 任务中，提示词中的不同组件（名称、属性、结构）模拟了对比学习中的序列微调步骤。
    *   **自适应难度调节**：引入动态置信度阈值 $\eta$。系统会根据前 10 个样本的平均置信度动态调整门控，决定实体是否进入下一阶段。
    *   **三阶段渐进式推理（课程学习）**：
        1.  **阶段 I（简单实体）**：仅使用实体名称，利用 LLM 内置知识快速对齐。若置信度高则直接输出。
        2.  **阶段 II（歧义实体）**：引入局部图结构（邻居信息）和训练集中的对齐示例，增强模型对歧义实体的辨析力。
        3.  **阶段 III（复杂实体）**：注入完整的多模态证据（图像相似度、关系三元组等），并采用“先思考再反思”的评分机制进行精细化对齐。

## 3. 实验设计
*   **数据集**：使用了 5 个公开基准数据集，包括 **DBP15K**（ZH-EN, JA-EN, FR-EN 三个子集）以及更具异构性的 **ICWIKI** 和 **ICYAGO**。
*   **基准模型（Benchmarks）**：对比了 20 种方法，涵盖：
    *   **表示学习类**：EVA, MCLEA, MEAformer, Simple-HHEA 等。
    *   **LLM 重排序类**：ChatEA, MM-ChatAlign, Seg-Align 等。
*   **评价指标**：Hits@1, Hits@10, MRR。

## 4. 资源与算力

*   **模型调用**：LLM 推理通过 API 完成，测试了 **Qwen2.5 (14B, 32B, 72B)** 和 **DeepSeek-v3**。
*   **效率提升**：在 ICWIKI 数据集上，PTFEA 将 Qwen2.5-72B 的运行时间从 21 小时缩短至 **1 小时**，Token 消耗降低了 **80% 以上**。

## 5. 实验数量与充分性
*   **实验规模**：涵盖了 5 个数据集的完整对比实验、针对不同规模 LLM 的性能分析、消融实验（验证三个阶段的必要性）、Token 和时间成本分析、以及针对缺失模态的鲁棒性测试。
*   **充分性评价**：实验设计非常充分。作者不仅验证了精度提升，还重点证明了该方法能显著缩小 14B 模型与 72B 模型之间的性能差距（差距从 30%+ 缩小到 0.6%），这对于实际应用具有极高的参考价值。

## 6. 主要结论与发现
*   **性能领先**：PTFEA 在所有数据集上均达到 SOTA，尤其在处理异构性强的 ICWIKI/ICYAGO 时优势明显。
*   **理论验证**：通过实验观察到双模型预测与注意力层输出的 $L_2$ 距离随训练步数增加而趋于零，验证了“提示即微调”的理论。
*   **规模鲁棒性**：课程学习策略使得小参数模型也能通过逐步获取信息达到大模型的对齐水平。
*   **极高效率**：通过提前过滤简单实体，避免了对绝大多数样本进行复杂的多模态推理，极大地节省了资源。

## 7. 优点
*   **理论深度**：首次为 MMEA 领域的上下文工程提供了数学解释，将黑盒操作转化为可解释的微调模拟。
*   **架构优雅**：课程学习的引入非常契合实体对齐任务中“证据难度不一”的本质特征。
*   **实用性强**：大幅降低了 LLM 在工业级知识图谱对齐任务中的落地门槛（时间和金钱成本）。

## 8. 不足与局限
*   **Hits@10 的微小下滑**：在某些数据集上，阶段 III 的 LLM 评分机制可能引入噪声，导致 Hits@10 略低于纯嵌入模型。
*   **依赖初始检索**：框架仍依赖第一阶段的候选列表生成，若初始检索召回率不足，后续 LLM 无法修正。
*   **API 依赖**：虽然降低了 Token 消耗，但核心推理仍依赖高性能 LLM API，对于完全私有化部署的超大规模图谱可能仍有压力。

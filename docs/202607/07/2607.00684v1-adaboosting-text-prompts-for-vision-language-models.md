## 摘要
预训练视觉语言模型（VLM）的分类准确率依赖于文本提示词的质量。手工设计的模板和大型语言模型（LLM）生成的描述不仅使预测更具可解释性，还使得相同的提示词能够在异构 VLM 之间复用。近期的研究利用少量标注图像构建任务自适应的文本提示词。然而，现有的少样本文本提示方法在构建提示词时并未显式关注被错误分类的样本，导致即使在样本数量增加时，性能提升也微乎其微。为了充分利用少样本监督，我们提出了文本提示增强（Text Prompt Boosting, TPB），这是一个受 AdaBoost 启发的框架，它将每个基于文本提示的分类器视为弱学习器，并通过显式针对困难的、被错误分类的样本，将它们顺序聚合为一个强集成模型。大量实验表明，TPB 在文本空间中保留了任务固有且与模型无关的线索，从而实现了鲁棒的跨模型迁移。在 11 个分类基准测试中，TPB 提高了源模型的准确率，并在迁移到更大、更强大的 VLM 时保留了由样本驱动的增益，而现有方法在这些情况下难以维持此类改进。

## Abstract
The classification accuracy of pretrained Vision-Language Models (VLMs) relies on the quality of the text prompts. Handcrafted templates and Large Language Model (LLM)-generated descriptions not only make predictions more interpretable, but also enable reuse of the same prompts across heterogeneous VLMs. Recent works construct task-adapted text prompts with a small number of labeled images. However, existing few-shot text prompting methods do not explicitly focus on misclassified examples during prompt construction, leading to only marginal improvements even as more shots become available. To fully exploit few-shot supervision, we propose Text Prompt Boosting (TPB), an AdaBoost-inspired framework that treats each text-prompt-based classifier as a weak learner and sequentially aggregates them into a strong ensemble by explicitly targeting hard, misclassified examples. Extensive experiments show that TPB preserves task-intrinsic, model-agnostic cues in text space, enabling robust cross-model transfer. Across eleven classification benchmarks, TPB improves accuracy on the source model and preserves shot-driven gains when transferred to larger, more capable VLMs, where existing methods struggle to sustain such improvements.

### 2. 方法论
TPB 将每个基于文本提示词的分类器视为一个“弱学习器”，通过多个回合的迭代构建一个“强集成模型”。
*   **核心思想**：利用 AdaBoost 机制，在每一轮迭代中增加被误分类样本的权重，迫使下一轮生成的提示词去解决这些困难案例。
*   **关键技术细节**：
    *   **大规模提示词池 (Large Prompt Pools)**：结合了手工模板、LLM 生成的描述以及它们的组合，形成丰富的候选空间。
    *   **贪婪提示词组合 (GPC, Greedy Prompt Composition)**：作为弱学习算法。分为两个阶段：1) **单模板初始化**：选择表现最好的共享模板；2) **类级别贪婪插入**：在保持离散空间的前提下，通过重复添加提示词（模拟连续权重）来最小化加权误差。
    *   **增强循环与图像增强**：在少样本设置下，为了防止过拟合，TPB 在每一轮对训练集进行图像增强（如随机裁剪、翻转），确保 Boosting 过程能持续获得有效信号。
    *   **SAMME.R 算法**：采用 AdaBoost 的多分类实值扩展版本进行样本权重更新和最终预测聚合。

### 6. 主要结论与发现
*   **卓越的扩展性**：TPB 在源模型上的表现随 Shot 增加而显著提升，克服了现有文本提示词方法性能过早饱和的问题。
*   **强大的迁移能力**：TPB 生成的自然语言提示词集成在迁移到更大、更强的模型时，能够保留在源模型上获得的性能增益，而基于梯度的优化方法（如 PEZ）在迁移时往往会失效。
*   **困难样本攻克**：定性分析证明，TPB 确实能够通过后续回合引入针对特定视觉特征（如遮挡、姿态）的描述来纠正之前的错误。

### 7. 优点（亮点）
*   **模型无关性**：由于输出的是纯文本提示词，它天然支持跨架构迁移，无需像软提示词（CoOp）那样进行复杂的映射。
*   **可解释性**：集成的提示词是人类可读的自然语言，便于理解模型关注的视觉特征。
*   **算法创新**：成功将经典的 AdaBoost 逻辑引入到离散的文本提示词搜索空间中，解决了少样本学习中的效率问题。

### 8. 不足与局限
*   **构建成本**：在大规模数据集或极大的候选提示词池上，贪婪搜索（GPC）的计算开销可能较大。
*   **噪声敏感性**：AdaBoost 机制可能会过度关注训练集中的噪声样本或异常值，导致在某些极端情况下过拟合。
*   **提示词池依赖**：性能受限于初始提示词池的质量，如果池中缺乏描述细粒度差异的文本，模型提升空间会受限。

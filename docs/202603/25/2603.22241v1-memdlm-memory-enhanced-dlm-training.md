## 摘要
扩散语言模型 (DLMs) 相比自回归 (AR) 模型具有显著优势，例如全注意力并行解码和灵活的生成能力。然而，它们面临显著的训练与推理不匹配问题：DLM 使用静态的单步掩码预测目标进行训练，但在部署时却采用多步渐进式去噪轨迹。我们提出了 MemDLM（记忆增强型 DLM），通过双层优化 (Bi-level Optimization) 将模拟去噪过程嵌入到训练中，从而缩小了这一差距。内层循环更新一组快速权重，形成参数化记忆 (Parametric Memory) 以捕捉每个样本的局部轨迹经验，而外层循环则在以此记忆为条件的情况下更新基础模型。通过将记忆压力从标记 (token) 表示转移到参数上，MemDLM 实现了更快的收敛速度和更低的训练损失。此外，内层循环可以在推理时作为自适应步骤重新启用，从而在长上下文理解方面获得额外收益。我们发现，当在推理时激活时，这种参数化记忆充当了一种涌现的权重内检索机制，帮助 MemDLM 在具有挑战性的“大海捞针” (Needle-in-a-Haystack) 检索任务中进一步减少标记级注意力瓶颈。代码：https://github.com/JarvisPei/MemDLM。

## Abstract
Diffusion Language Models (DLMs) offer attractive advantages over Auto-Regressive (AR) models, such as full-attention parallel decoding and flexible generation. However, they suffer from a notable train-inference mismatch: DLMs are trained with a static, single-step masked prediction objective, but deployed through a multi-step progressive denoising trajectory. We propose MemDLM (Memory-Enhanced DLM), which narrows this gap by embedding a simulated denoising process into training via Bi-level Optimization. An inner loop updates a set of fast weights, forming a Parametric Memory that captures the local trajectory experience of each sample, while an outer loop updates the base model conditioned on this memory. By offloading memorization pressure from token representations to parameters, MemDLM yields faster convergence and lower training loss. Moreover, the inner loop can be re-enabled at inference time as an adaptation step, yielding additional gains on long-context understanding. We find that, when activated at inference time, this Parametric Memory acts as an emergent in-weight retrieval mechanism, helping MemDLM further reduce token-level attention bottlenecks on challenging Needle-in-a-Haystack retrieval tasks. ### 论文总结：MemDLM: Memory-Enhanced DLM Training

#### 1. 核心问题与整体含义（研究动机和背景）
扩散语言模型（DLMs）虽然具备并行解码和双向上下文感知的优势，但面临严重的**训练-推理不匹配（Train-Inference Mismatch）**问题。
*   **现状**：标准 DLM 在训练时采用静态单步掩码预测（给定 ground-truth 文本，随机掩码后预测原词）；而在推理时，模型需通过多步迭代去噪生成文本，每一步都依赖于前一步可能包含错误的预测结果。
*   **痛点**：这种不匹配导致了**暴露偏差（Exposure Bias）**。模型在推理过程中过度依赖脆弱的 Token 表示来维持上下文，一旦产生预测误差，错误会迅速累积，导致长文本处理能力下降。
*   **动机**：通过引入“参数化记忆”，将局部去噪轨迹的经验内化到模型参数中，从而减轻 Token 表示的记忆压力，弥合训推差距。

#### 2. 方法论：核心思想与技术细节
MemDLM 提出了一种基于**双层优化（Bi-level Optimization）**的训练框架：
*   **核心思想**：在训练的每一个步骤中，模拟一个简化的去噪过程。利用“快速权重”（Fast Weights）捕捉当前样本的局部特征，再以此为基础更新主模型。
*   **内循环（Inner Loop - 记忆形成）**：
    *   使用参数高效的 LoRA 插件作为快速权重 $\phi$。
    *   **锚点一致轨迹（Anchor-Consistent Trajectory）**：分为两个阶段。第一阶段（Pre-Anchor）将当前状态进一步掩码，让模型学习从更嘈杂的状态恢复到当前状态；第二阶段（Anchor-to-Target）学习从当前状态恢复到原始清洁文本。
    *   通过梯度下降更新 $\phi$，使其成为捕捉该样本局部轨迹经验的“参数化记忆”。
*   **外循环（Outer Loop - 基础模型更新）**：
    *   在已知参数化记忆 $\phi$ 的条件下，计算基础模型 $\theta$ 的去噪损失并进行更新。
    *   采用一阶近似（First-Order Approximation）以避免计算复杂的二阶 Hessian 矩阵，提高训练效率。
*   **推理阶段自适应**：推理时可选择性地重新启用内循环，将长 Prompt 的信息通过梯度更新直接“写入”权重，实现一种**权重内检索（In-weight Retrieval）**机制。

#### 6. 主要结论与发现

*   **收敛更优**：MemDLM 在训练过程中表现出更快的收敛速度和更低的验证集损失，说明参数化记忆降低了基础模型的优化难度。
*   **外推能力**：即使在超出训练长度（如 16K、32K）的情况下，MemDLM 依然保持了对基准模型的领先优势。
*   **机制发现**：推理时的内循环更新起到了“权重内检索”的作用，能有效缓解 Token 级注意力的瓶颈。

#### 7. 优点
*   **理论创新**：首次将双层优化引入 DLM 训练以解决训推不匹配问题，思路清晰且针对性强。
*   **架构解耦**：参数化记忆可以作为一种可选的推理插件，不改变模型推理的基本逻辑，且在不开启推理自适应时依然有显著增益。
*   **鲁棒性**：通过将信息存储在参数空间而非易受损的 Token 空间，增强了模型对去噪过程中噪声累积的抵抗力。

#### 8. 不足与局限
*   **训练开销**：尽管进行了优化，但双层优化本质上增加了前向和后向传播的次数，训练成本高于标准 MDLM。
